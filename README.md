# <img src="assets/images/cat.svg" width="32"> Lionizon
Roblox quality of life extension with expanding features.

## Features

### 🏠 Home Page
- **Welcome banner** -> Displays your avatar, username, and online status with a colored presence ring on the home page
- **Presence circles** -> Shows colored border rings on friend cards indicating if they're in-game (green), online (blue), or in Studio (orange)
- **Friend server location** -> Hovering over a friend's card shows the city and country of the server they're playing on

### 🎮 Game Pages
- **Server geolocation** -> Each server card displays the location of the server
- **Server ping & FPS** -> Displays ping and FPS for every server
- **Performant server filter** -> Sort all servers by a combined ping + FPS score to find the best one almost instantly
- **Playtime heatmap** -> Shows a GitHub-style activity heatmap of your daily playtime for each game
- **Live game stats** -> View player count, visits, and favorites counters of any experience in real time
- **Subplaces tab** -> Adds a dedicated tab listing all subplaces of a game with thumbnails and quick-join buttons
- **Fast join** -> Play buttons on game cards that launch directly into a game instance

### 💰 Robux
- **Robux to currency converter** -> Converts your Robux balance to your chosen real-world currency, displayed next to the Robux icon

### ⚙️ Settings
- **Lionizon settings page** -> Dedicated settings tab injected into Roblox's account settings page
- Currency selector (30+ currencies supported)
